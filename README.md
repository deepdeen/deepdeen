# DeepDeen
Islamic Knowledge Infrastructure — Unified API for Quran, Hadith, Tafsir & Isnad

## What is DeepDeen?
DeepDeen is the central data infrastructure for Islamic knowledge in the AI era. 
A unified, structured, and queryable API covering the Quran, Hadith with full 
Isnad chains, Tafsir, Fiqh, and classical Islamic philosophical texts.

## Status
🚧 Under active development — Phase 1

## Stack
- Supabase (PostgreSQL + pgvector)
- Neo4j (Isnad graph)
- Next.js + Vercel
- AraBERT (NLP)
